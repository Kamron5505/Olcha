# Руководство по использованию админ-панели Olcha

## 🚀 Быстрый старт

### 1. Доступ к админ-панели
**URL:** `http://localhost:5173/admin/login`

**Тестовые учетные данные:**
- **Email:** `admin@olcha.uz`
- **Пароль:** `admin123`

### 2. Основные разделы
1. **Дашборд** - Общая статистика и быстрый доступ
2. **Заказы** - Управление заказами клиентов
3. **Товары** - Управление каталогом товаров
4. **Категории** - Управление категориями товаров
5. **Пользователи** - Управление пользователями
6. **Настройки** - Настройки магазина

## 📊 Дашборд

### Что показывает:
- **Всего заказов** - общее количество заказов
- **Общая выручка** - сумма всех продаж
- **Товары** - количество товаров в каталоге
- **Пользователи** - количество зарегистрированных пользователей
- **Последние заказы** - 5 последних заказов
- **Товары с низким запасом** - товары, которые скоро закончатся

### Быстрые действия:
- **Добавить товар** - быстрый переход к созданию нового товара
- **Добавить категорию** - создание новой категории

## 📦 Управление заказами

### Просмотр заказов
1. Перейдите в раздел **"Заказы"**
2. Используйте фильтры для поиска:
   - **Статус** - фильтр по статусу заказа
   - **Сортировка** - по дате или сумме
   - **Поиск** - по ID заказа или имени клиента

### Детали заказа
Нажмите **"Подробнее"** на любом заказе, чтобы увидеть:
- Полную информацию о заказе
- Состав заказа (товары, количество, цены)
- Информацию о клиенте
- Способ доставки и оплаты

### Изменение статуса заказа
1. В деталях заказа выберите новый статус из выпадающего списка
2. Статусы:
   - **Ожидает** - новый заказ
   - **В обработке** - заказ подтвержден, готовится к отправке
   - **Отправлен** - заказ передан в службу доставки
   - **Доставлен** - заказ получен клиентом
   - **Отменен** - заказ отменен

### Действия с заказами
- **Уведомить клиента** - отправить email/SMS уведомление
- **Распечатать** - печать деталей заказа
- **Удалить** - удаление заказа (только для отмененных)

## 🛍️ Управление товарами

### Просмотр товаров
1. Перейдите в раздел **"Товары"**
2. Используйте фильтры:
   - **Категория** - фильтр по категории товара
   - **Статус** - активные, нет в наличии, черновики
   - **Поиск** - по названию или ID товара

### Добавление нового товара
1. Нажмите **"+ Добавить товар"**
2. Заполните форму:
   - **Название товара** - обязательное поле
   - **Цена** - цена в сумах
   - **Категория** - выбор из существующих категорий
   - **Количество на складе** - доступное количество
   - **Статус** - активен, нет в наличии, черновик
   - **Описание** - подробное описание товара
3. Нажмите **"Создать товар"**

### Редактирование товара
1. Нажмите **"Редактировать"** на карточке товара
2. Внесите изменения
3. Нажмите **"Сохранить изменения"**

### Управление изображениями
- После создания товара можно добавить изображения
- Поддерживается несколько изображений для одного товара
- Первое изображение используется как основное

### Статусы товаров
- **Активен** - товар отображается на сайте, доступен для покупки
- **Нет в наличии** - товар отображается, но нельзя купить
- **Черновик** - товар не отображается на сайте

## 📁 Управление категориями

### Создание категории
1. Нажмите **"Добавить категорию"** в быстрых действиях
2. Укажите:
   - **Название категории** - например, "Смартфоны"
   - **URL-слаг** - автоматически генерируется из названия
3. Нажмите **"Создать категорию"**

### Редактирование категорий
- Изменение названия категории
- Обновление URL-слага
- Удаление категорий (только если в них нет товаров)

## 👥 Управление пользователями

### Типы пользователей
- **Администратор** - полный доступ ко всем функциям
- **Менеджер** - может управлять заказами и товарами
- **Пользователь** - обычный покупатель

### Изменение ролей
1. Найдите пользователя в списке
2. Измените роль в выпадающем списке
3. Изменения применяются автоматически

## ⚙️ Настройки

### Основные настройки
- **Название сайта** - отображается в шапке сайта
- **Валюта** - валюта для отображения цен (UZS)
- **Стоимость доставки** - цена доставки после бесплатного лимита
- **Бесплатная доставка** - количество бесплатных доставок
- **Режим обслуживания** - временное отключение сайта

### Сохранение настроек
Все изменения сохраняются автоматически при нажатии **"Сохранить"**

## 🔧 Технические особенности

### Mock API

- При перезагрузке страницы данные сохраняются
- Для сброса данных очистите `localStorage`

## 🚨 Устранение неполадок

### Проблема: Не могу войти в админ-панель
**Решение:**
1. Проверьте правильность email и пароля
2. Очистите кэш браузера
3. Проверьте консоль на наличие ошибок JavaScript

### Проблема: Данные не сохраняются
**Решение:**
1. Проверьте, включены ли cookies в браузере
2. Убедитесь, что `localStorage` доступен
3. Проверьте консоль на ошибки

### Проблема: Изображения не загружаются
**Решение:**
1. Проверьте интернет-соединение
2. Убедитесь, что URL изображений корректные
3. Используйте изображения с поддерживаемых доменов

## 📈 Советы по использованию

### Для эффективного управления:
1. **Регулярно обновляйте статусы заказов** - клиенты видят актуальную информацию
2. **Следите за остатками товаров** - избегайте ситуаций "нет в наличии"
3. **Используйте категории** - упрощает навигацию для покупателей
4. **Анализируйте статистику** - понимайте, какие товары популярны

### Для безопасности:
1. **Используйте сложные пароли** - минимум 8 символов, буквы и цифры
2. **Регулярно меняйте пароли** - каждые 3 месяца
3. **Ограничивайте доступ** - только доверенным сотрудникам
4. **Ведите журнал действий** - отслеживайте изменения
