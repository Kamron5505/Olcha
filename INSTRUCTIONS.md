# Инструкция по запуску и использованию

## Установка и запуск

1. **Установите зависимости:**
```bash
cd olcha-clone
npm install
```

2. **Запустите приложение:**
```bash
npm run dev
```

3. **Откройте в браузере:**
```
http://localhost:5173
```

## Тестирование новых функций

### 1. Быстрый просмотр товара
1. Перейдите на главную страницу или в каталог
2. Наведите курсор на карточку товара
3. Нажмите кнопку "Быстрый просмотр"
4. Проверьте работу модального окна

### 2. Оформление заказа
1. Добавьте товары в корзину
2. Перейдите в корзину
3. Нажмите "Оформить заказ"
4. Заполните форму:
   - Контактные данные
   - Выберите способ получения (самовывоз или доставка)
   - Выберите способ оплаты
5. Подтвердите заказ
6. Проверьте страницу успешного оформления

### 3. Админ-панель
1. Нажмите на иконку профиля в шапке
2. Выберите "Админ-панель"
3. Войдите с учетными данными:
   - Логин: `admin`
   - Пароль: `admin123`
4. Проверьте функционал:
   - Просмотр заказов
   - Фильтрация по статусу
   - Изменение статуса заказа
   - Удаление заказов

## Особенности реализации

### Доставка
- Первые 3 доставки бесплатно
- Начиная с 4-й доставки - 20,000 сум
- Счетчик сохраняется в localStorage

### Хранение данных
- Заказы сохраняются в localStorage
- При перезагрузке страницы данные сохраняются
- В реальном приложении нужно заменить на серверное API

### Безопасность
- Админ-панель защищена простой авторизацией
- В продакшене нужна серверная аутентификация
- Пароли должны храниться в хэшированном виде

## Проблемы и решения

### Если не работает быстрый просмотр:
- Проверьте, что компонент ProductModal импортирован
- Убедитесь, что кнопка "Быстрый просмотр" появляется при наведении
- Проверьте консоль браузера на наличие ошибок

### Если не работает оформление заказа:
- Проверьте, что корзина не пуста
- Убедитесь, что все обязательные поля заполнены
- Проверьте localStorage на наличие заказов

### Если не работает админ-панель:
- Проверьте авторизацию (логин: admin, пароль: admin123)
- Убедитесь, что в localStorage есть заказы
- Проверьте консоль на наличие ошибок JavaScript
